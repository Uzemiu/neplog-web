# Neplog-web

这是一个基于Springboot + Hibernate + Redis + Vue开发的轻量级前后端分离个人博客项目的前端部分。
前端组件部分主要基于ElementUI。
同时也是一个个人以学习为主要目的的项目，主要是对以上所有技术的使用层面上的熟悉。

后端仓库见[neplog](https://github.com/Uzemiu/neplog)

目前项目正在开发中...
